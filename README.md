**Simple Note Taking App**

📝 Notes App – Express & MySQL
📌 Overview

This is a simple Notes Taking API built using Node.js, Express.js, and MySQL. The application provides basic CRUD (Create, Read, Update, Delete) operations to manage notes efficiently.

This project is my first experience working with a relational database (MySQL), where I learned how to connect a backend server with a database, write SQL queries, and structure APIs.

🚀 Features
Create a new note
Fetch all notes
Fetch a single note by ID
Update an existing note
Delete a note
RESTful API design
🧰 Tech Stack
Backend: Node.js, Express.js
Database: MySQL
API Testing: Postman / Thunder Client
